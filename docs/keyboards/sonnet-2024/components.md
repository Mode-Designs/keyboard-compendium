## Case Compatibility

Top and bottom cases for the Mode Sonnet are interchangeable across generations, but the combination limits which mounting styles are available.

| Bottom ↓ / Top → | Sonnet 2022 | Sonnet 2024 | Sonnet 2026 |
|:---|:---|:---|:---|
| **Sonnet 2022** | Iso Top Mount, Stack Mount | Iso Top Mount | Iso Top Mount |
| **Sonnet 2024** | Iso Top Mount, Stack Mount | Iso Top Mount, Lattice Block Mount, Top Mount Block | Iso Top Mount, Lattice Block Mount, Top Mount Block |
| **Sonnet 2026** | Iso Top Mount | Iso Top Mount, Lattice Block Mount, Top Mount Block | Iso Top Mount, Lattice Block Mount, Top Mount Block |
