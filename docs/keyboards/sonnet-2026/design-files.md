## Plate

The plate and PCB have to be chosen together on this board. The hotswap PCB ([75% PCB / Hotswap M75H-V2](../../parts/pcb-m75h-v2.md)) supports ANSI only, so it pairs with the ANSI plate. An ISO build requires the solder PCB ([75% PCB /Solder M75S-V2](../../parts/pcb-m75s-v2.md)) together with the ISO plate, which Mode does not sell; use the files below to have it made.
