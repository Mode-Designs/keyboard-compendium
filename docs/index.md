# Overview

The Mode Keyboard Compendium is the definitive reference for keeping Mode keyboards on the desk for as long as possible — part information, cross-compatibility, availability, and design files for every board we've made.

## How to use this site

1. Find your board under **Keyboards** in the navigation.
2. Open its **Components** page for the full parts list, compatibility, and availability.
3. Follow the part links for more detail, design files, and sourcing options.

Each keyboard also includes:

- **Design Files** — CAD files for making replacement parts, or to use as the basis for a customized part.
- **Community Projects** — vetted community-created projects, modifications, and resources.
