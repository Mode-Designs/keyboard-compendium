# About

The Mode Keyboard Compendium exists to make Mode keyboards extensible and maintainable for their owners long term.

Keyboards leave production, but they shouldn't leave desks. This site documents legacy and current boards: which parts fit which boards, what's still available from Mode, what can be sourced from third parties, and which parts you can have made yourself using the provided design files. It also serves as a directory of vetted community projects and resources.

Design files are provided for personal, non-commercial use only — see the license in the repository for details.

For anything this site doesn't answer, contact [support@modedesigns.com](mailto:support@modedesigns.com).
